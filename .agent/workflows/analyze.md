# /analyze - Marketing Analytics Workflow

Follow this workflow to analyze marketing performance.

## Step 1: Define Scope

1. **Time period**: What dates to analyze?
2. **Channels**: Which channels?
3. **Campaigns**: Specific campaigns?
4. **Goals**: What metrics matter?

## Step 2: Gather Data

Collect data from:

1. **Web analytics**: GA4, similar
2. **Ad platforms**: Google, Meta, LinkedIn
3. **Email**: ESP metrics
4. **Social**: Platform analytics
5. **CRM**: Lead/sales data

## Step 3: Analyze Metrics

### Traffic Analysis

| Metric | Check |
|--------|-------|
| Sessions | Volume and trend |
| Sources | Channel breakdown |
| Bounce rate | Quality signals |
| Engagement | Time, pages |

### Conversion Analysis

| Metric | Check |
|--------|-------|
| Conversion rate | By source, campaign |
| Cost per conversion | Efficiency |
| Funnel drop-off | Where we lose people |

### Revenue Analysis

| Metric | Check |
|--------|-------|
| Revenue | By channel, campaign |
| ROAS | Return on ad spend |
| LTV | Customer value |
| CAC | Acquisition cost |

## Step 4: Generate Insights

1. **What's working**: Top performers
2. **What's not**: Underperformers
3. **Why**: Root causes
4. **Trends**: Patterns over time

## Step 5: Recommend Actions

1. **Quick wins**: Easy improvements
2. **Optimizations**: Tune what's working
3. **Tests**: Experiments to run
4. **Cuts**: Stop what's not working

## Step 6: Report

Create actionable report:

1. **Executive summary**: Key takeaways
2. **Performance**: Metrics vs goals
3. **Insights**: Why it happened
4. **Recommendations**: What to do next

## Output

- Analytics report
- Dashboard updates
- Action items
- Test hypotheses
